## About This Repository

This repository contains multiple standalone mini-projects inside the `Projects/` directory.
Each project is independent and has its own structure (HTML, CSS, JS, and assets where needed).

It is useful for:

- Frontend practice and revision
- UI/UX layout experimentation
- Portfolio presentation
- Beginner JavaScript interaction projects

## Run Locally

1. Clone or download this repository.
2. Open the project root in VS Code.
3. Go to any folder inside `Projects/`.
4. Open that project's `index.html` in a browser.
5. Optional: Use the Live Server extension for faster preview.

## Notes

- All projects are self-contained.
- Asset folders may vary by project (`images`, `icons`, `photo`, etc.).
- This repository is focused on learning and showcasing frontend fundamentals.
